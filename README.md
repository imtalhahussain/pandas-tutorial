## Pandas - Pandas is an open source library designed for data manipulation and analysis. it makes working with structured data fast, flexible and intuitive - especially if you are dealing with CSV files, Execel sheets, SQL tables, JSONs and APIs.
# - The two core data structure in Pandas are:
# 1. Series : A 1D labeled array (like column)
  2. DataFrame : A 2D labeled data structure (like a full spread sheet or SQL table)

## DATAFRAME : A two dimensional, tabular data structure with labeled rows and columns. Think of it as a Python -native spreadsheet - but way more powerful and programmable.

# key features:
. Columnn wise and row wise operations.
. Support for mixed data types.
. Fast, vectorized operations (built on Numpy).
